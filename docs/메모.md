- 스트림 상태

  > 네임 스페이스 : ios_base
  >
  > - 비트 플래그
  >
  >   ```
  >   goodbit
  >   eofbit
  >   failbit
  >   badbit
  >   ```
  >
  > - 메소드 버전
  >
  >   ```
  >   good()
  >   eof()
  >   fail()
  >   bad()
  >   ```
  >
  > - 사용 예
  >
  >   ```cpp
  >   	// Stream States : 스트림 상태
  >   	// 데이터를 읽을 때 실패했는지 성공했는지 확인할 수 있도록 저장되는 상태
  >   	char line[5] = "aaa";
  >   	cin >> line;
  >   	// eof : end of file
  >   	if (!cin.eof()) {
  >   		...
  >   	}
  >   ```
